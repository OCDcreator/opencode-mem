# What OpenCode Memory Does

OpenCode Memory is a long-term memory plugin for OpenCode.

Its job is simple: keep the useful parts of your AI collaboration so you can come back later without re-explaining the same project background, decisions, fixes, and working rules.

## What it can help you do

- Save valuable work automatically
- Search past solutions and project decisions
- Save a memory manually when you want to keep something right now
- Review your user profile and working habits
- Configure the provider and embedding setup that power summaries and search
- Use either remote embeddings or local embedding models for search
- Use global config or override settings per project
