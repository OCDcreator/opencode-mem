# How to Use the Project Memories Page

The Project Memories page is the most important page in the plugin.

You can think of it as the history and knowledge page for the current project.

## What you will see here

### Memories

These are the cleaned-up results produced by the plugin.

They are usually shorter and easier to reuse than the original conversation.

### Prompts

These are the original user messages that started the interaction.

Sometimes a prompt and a memory are shown together so you can see both the question and the outcome.

## What you can do here

- Search past memories
- Filter by tag
- Open detailed memory content
- Pin important memories
- Edit memories
- Delete one or many items

## Common ways to use this page

### Remember how something was solved before

Search by error text, module name, or feature name.

### Recover project rules

Search by naming conventions, architecture terms, or important constraints.

### Prepare for a new task

Look up the project area first so the next AI session starts with better context.

If you only learn one page in this plugin, learn this one first.
